Ahora hablemos sobre tablas y funciones Hash Una tabla hash es una estructura de datos que maneja claves numericas su funcion principal se enfoca a realizar una busqueda de un dato entre mucha informacion aqui es donde entra la clave en accion pues esta permite el acceso al dato que buscamos en la tabla El uso de estas tablas agiliza el proceso de busqueda y se usa para almacenar grandes cantidades de informacion ya que siempre se trata de alcanzar un tiempo minimo para encontrar la informacion pero en el peor de los casos puede llegar mucho la busqueda se realizara en todos los lugares y el dato buscado sea el ultimo Para poder esparcir los datos en la tabla se hace uso de una clave generada por una funcion Hash o funcion picadillo esta es la encargada de realizar una dispersion de datos pareja donde cada lugar de la tabla sea usado y el tiempo de busqueda minimo o por lo menos se encarga de que la clave generada se repita lo menos posible porque entre menos colisiones mayor eficiencia Este tipo de funciones tiene de entrada por lo general cadenas de textos de longitud fija y su propiedad mas destacable es que pueda ser calculada facilmente y al mismo tiempo minimicen las colisiones claves repetidas de los elementos Un ejemplo de tabla Hash puede ser la tabla ASCII cada caracter tiene su valor en la tabla y el tiempo de busqueda es minima
